# life-timeline
生活記録管理アプリ

## 【開発メモ】

・データの持ち方設計から整理する

・1時間区切りで24要素のArray

・生成したカードの内容(作業名, 属性)を持たせる

・記録・保存だけどうする？無限にためていく？

・まずはテキストやコンソールから入力できるようにして、その後表示・ドラッグなど整える


## 【開発メモ(旧)】

・将来的にはWeb（React）がいい？→触ってみたけどいまいち

・縦軸が曜日、横軸が0-24時のボックス形式

・気軽に入力できる(食事・掃除などのカードをドラッグ&ドロップ？)

・カードは自由な項目を追加可能

・カードの属性(睡眠・作業・遊び・生活など)ごとの平均時間を表示

・最大で一か月分までまとめて表示
