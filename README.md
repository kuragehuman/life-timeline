# life-timeline
生活記録管理アプリ

## 【開発メモ】

・データの持ち方設計から整理する

・1時間区切りで24要素のArray

・生成したカードの内容(作業名, 属性)を持たせる

・記録・保存だけどうする？無限にためていく？

・まずはテキストやコンソールから入力できるようにして、その後表示・ドラッグなど整える


・7日24時間(1時間ごと)で7*24の要素を持つArrayを用意

・propertyはevent, typeの二つ、デフォルト値は"~~~~"と"-1"

## 【仮UI】

・コンソールで"add (曜日) (開始時刻) (終了時刻) (名称)"を入力するとその時間帯に当たるArray要素のeventプロパティにその名称を追加

・コンソールで"print"を入力すると7*24のArrayの表を表示する(テキストだが線を入れてみやすいように)



## 【開発メモ(旧)】

・将来的にはWeb（React）がいい？→触ってみたけどいまいち

・縦軸が曜日、横軸が0-24時のボックス形式

・気軽に入力できる(食事・掃除などのカードをドラッグ&ドロップ？)

・カードは自由な項目を追加可能

・カードの属性(睡眠・作業・遊び・生活など)ごとの平均時間を表示

・最大で一か月分までまとめて表示
